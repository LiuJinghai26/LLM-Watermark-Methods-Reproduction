| **A Watermark for Large Language Models**                         | 主要实验使用 **OPT-1.3B** 作为生成模型；图 1 示例中使用 **OPT-6.7B**；还用 **OPT-2.7B** 作为 oracle model 来计算生成文本的 perplexity。                                                               | 使用 **C4 数据集的 news-like / RealNewsLike 子集**。做法是把文本前半部分当 prompt，截取后面的固定长度 token 作为 human baseline completion，再让模型生成对应长度的文本。实验通常生成约 **500 个样本**，长度约 **200 tokens**。                 |

| **Who Wrote this Code? Watermarking for Code Generation / SWEET** | 主实验使用 **StarCoder** 作为代码生成模型；附录还测试 **LLaMA2**；ClassEval 任务中使用 **StarCoder-Instruct**。检测/对比基线包括 **WLLM、EXP-EDIT、DetectGPT、GPTZero、OpenAI Classifier、LOGP、LOGRANK** 等。 | 主数据集是 **HumanEval、MBPP、DS-1000**，都是 Python 代码生成任务；扩展实验还用了 **HumanEvalPack** 的 C++ / Java 版本，以及 **ClassEval** 的类级 Python 代码生成任务。HumanEval 有 164 题，MBPP 有 500 题，DS-1000 有 1000 题。  |
